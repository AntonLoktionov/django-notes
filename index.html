Разница между values и values_list Django

values() возвращает QuerySet, содержащий словари: <QuerySet [{'comment_id': 1}, {'comment_id': 2}]>values_list() возвращает QuerySet с кортежами: <QuerySet [(1,), (2,)]>При использовании в values_list() одного поля, можно использовать параметр flat=True, чтобы вернуть QuerySet со списком значений вместо кортежей. <QuerySet [1, 2]>

Разница между aggregate и annotate Django ORM

Разница между aggregate и annotate в том, что aggregate вычисляет функцию для всего набора данных и возвращает словарь (dict) с вычисленным значениемannotate вычисляет значения для каждой записи набора данных и возвращает QuerySet
